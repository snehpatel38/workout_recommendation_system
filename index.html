💪 Workout Recommendation System

Personalized Fitness Workout Recommender

This system provides personalized workout suggestions based on user input like age, fitness goals, and preferences. It uses similarity metrics and filters to return workouts best suited to the user’s needs.

📌 Features

  • Recommends workouts using cosine similarity
  • Filters by body part, equipment, and level
  • Built using Flask (Python web framework)
  • Dockerized for easy deployment and scalability

🧠 Technologies Used

  • Python
  • Flask for backend routing
  • Pandas and Scikit-learn
  • Cosine Similarity for recommendation logic
  • Docker & Docker Compose for containerization
