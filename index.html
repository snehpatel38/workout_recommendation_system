💪 Workout Recommendation System

Personalized Fitness Workout Recommender

This system provides personalized workout suggestions based on user input like age, fitness goals, and preferences.

📌 Features

  • Recommends workouts using cosine similarity
  • Filters by body part, equipment, and level
  • Built using Streamlit
  • Clean UI with easy-to-use interface

🧠 Technologies Used

  • Python
  • Scikit-learn, Pandas
  • Streamlit
  • Cosine Similarity for recommendations
